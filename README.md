# PiezoPower Analyzer

A Python-based data analysis dashboard for studying voltage generated by piezoelectric tiles. This project detects footsteps from voltage spikes, calculates experiment statistics, estimates capacitor energy storage, visualizes sensor data, and generates simple reports.

This project was built to connect **Electrical & Electronic Engineering**, **renewable energy harvesting**, and **Python data analysis**.

---

## Project Overview

Piezoelectric materials can generate electrical voltage when pressure is applied to them. In this project, voltage readings from piezoelectric tiles are analyzed to understand how much electrical output can be produced from human footsteps.

The dashboard takes voltage data from a CSV file and processes it to identify step events, calculate peak voltage, estimate energy stored in a capacitor, and display useful graphs for experiment analysis.

---

## Features

* Upload and analyze piezoelectric voltage data
* Detect footsteps from voltage spikes
* Count total steps from sensor readings
* Calculate peak voltage and average voltage
* Estimate capacitor energy using the formula:

[
E = \frac{1}{2}CV^2
]

* Display voltage graphs using Matplotlib
* Generate experiment summaries
* Export results as a PDF report
* Includes sample data for testing the dashboard

## How It Works

1. The user uploads a CSV file containing time and voltage readings.
2. The program reads the data using Pandas.
3. A voltage threshold is used to detect step events.
4. The dashboard calculates useful statistics such as:

   * Total steps
   * Maximum voltage
   * Average voltage
   * Estimated stored energy
5. Graphs are generated to visualize the voltage changes over time.
6. A PDF report can be exported for documentation.

## Sample CSV Format

The uploaded CSV file should follow this format:

| time_seconds | voltage |
| ------------ | ------- |
| 0.00         | 0.02    |
| 0.10         | 0.05    |
| 0.20         | 1.42    |
| 0.30         | 0.08    |

Example:

```csv
time_seconds,voltage
0.00,0.02
0.10,0.05
0.20,1.42
0.30,0.08
0.40,0.04
```

## Energy Calculation

The project estimates the energy stored in a capacitor using:

[
E = \frac{1}{2}CV^2
]

Where:

* **E** = energy stored in joules
* **C** = capacitance in farads
* **V** = voltage in volts

This helps estimate how much energy can be stored from the voltage generated by footsteps.
